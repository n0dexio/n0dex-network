# n0dex

> AI powered by people, not data centers.

n0dex is a decentralized AI compute network that enables anyone to share unused computing resources, earn rewards, and access private, uncensored intelligence without relying on centralized cloud providers.

## 🚀 Vision

The future of AI should not be controlled by a handful of data centers.

n0dex creates a peer-to-peer infrastructure where individuals contribute compute power and collectively build a global intelligence network that is:

* Decentralized
* Privacy-first
* Permissionless
* Censorship-resistant
* Community-powered

## 🌐 How It Works

### 1. Share Compute

Contribute your unused CPU or GPU resources to the network.

### 2. Earn Rewards

Receive incentives for providing computational power and helping process AI workloads.

### 3. Access Intelligence

Use decentralized AI services without depending on centralized providers or exposing sensitive data.

## 🏗️ Why n0dex?

Traditional AI depends on massive centralized data centers that control infrastructure, access, and policies.

n0dex introduces an alternative model where:

* People own the network
* Compute is distributed globally
* AI access becomes more open and resilient
* Contributors are rewarded for participation

## 📊 Ecosystem

n0dex connects three core participants:

| Participant        | Role                              |
| ------------------ | --------------------------------- |
| Compute Providers  | Share hardware resources          |
| AI Users           | Access decentralized intelligence |
| Network Validators | Help maintain network integrity   |
